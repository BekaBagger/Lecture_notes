# Финальный проект: Чат-бот «Lecture Notes»
## Дисциплина: «Программирование на Python»

## 1. Описание проекта
Данный проект представляет собой интерактивный Telegram-бот — сервис хранения учебных конспектов. Он позволяет автоматизировать процесс структурирования образовательных материалов, распределяя текстовые заметки по предметам, прикрепляя учебные файлы, добавляя теги и реализуя мгновенный полнотекстовый поиск по базе знаний.

## 2. Используемые технологии
* **Язык программирования:** Python 3.14
* **Библиотека API:** pyTelegramBotAPI (для построения пользовательского интерфейса)
* **База данных:** SQLite3 (для изоляции данных и логирования действий)
